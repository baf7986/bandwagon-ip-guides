## 换 IP 之后还需要做什么

很多人换完 IP 发现有些地方还是不对，大概是因为忘了做后续配置：

1. **更新 DNS 解析**：域名的 A 记录要改成新 IP，不然访客还是往旧地址跑
2. **更新 SSH 登录配置**：本地 SSH config 文件里如果有绑定旧 IP 的配置，要改过来
3. **重新测试一遍**：换完之后用新 IP ping 一下，确认可达，再试 SSH 连接

数据、网站文件、root 密码这些都不会动，服务器里面的东西完整保留。

## FAQ

**搬瓦工可以给一台 VPS 添加多个 IP 吗？**

不可以。搬瓦工每台 VPS 固定分配一个独立 IP，不支持单独购买额外 IP 地址。如果需要不同 IP，只能开多台服务器。
